Progress

caution

This is unreleased documentation for headless-styles package.

Progress is used to display the progress status for a task that takes a long time or consists of several steps.

View source

Import‚Äã

import { getProgressProps } from '@pluralsight/headless-styles'

Usage‚Äã

Basic Progress‚Äã

getProgressProps comes with two kinds to change the visual style of the progress. You can use linear (default), and inset.

const progress = getProgressProps({ now: 50 })

<div {...progress.wrapper}>
  <div {...progress.bar} />
</div>

Inset Progress‚Äã

The inset kind is to be used when pinned to the edges of a container.

const insetProgress = getProgressProps({ kind: 'inset', now: 50 })

<div {...insetProgress.wrapper}>
  <div {...insetProgress.bar} />
</div>

Progress sizes‚Äã

There are two sizes available for the progress. You can use xs or s (default).

const xsProgress = getProgressProps({ size: 'xs', now: 50 })
const progress = getProgressProps({ size: 's', now: 60 })

<div {...xsProgress.wrapper}>
  <div {...xsProgress.bar} />
</div>
<div {...progress.wrapper}>
  <div {...progress.bar} />
</div>

Accessibility‚Äã

getProgressProps returns an a11yProps Object to spread over the "bar" component.

  • It sets the role to progressbar to denote that it is a progress.
  • It uses aria-valuenow set to the now value passed into the ProgressOptions to ensure the progress percent is visible to screen readers.

CSS API‚Äã

getProgressProps‚Äã

getProgressProps(options: ProgressOptions): ProgressProps

The progress prop-getter takes in ProgressOptions returns ProgressProps that contains the a11y attributes needed along with all the styles for you to use in any fashion you like - or easily extend/overwrite when needed.

JS API‚Äã

getJSProgressProps‚Äã

getJSProgressProps(options: ProgressOptions): JSProgressProps

The CSS-in-JS prop-getter takes in the same ProgressOptions as the CSS API and returns JSProgressProps which caters to CSS Templates, StyleObject, and a11yProps.

note

The custom bar width percentage is added into the cssProps and styles for the JS API.

ProgressOptions‚Äã

This interface is available as an import for use.

interface ProgressOptions extends ProgressA11yOptions {
  kind?: 'linear' | 'inset'
  size?: 'xs' | 's'
  tech?: 'svelte'
}
Import example
import type { ProgressOptions } from '@pluralsight/headless-styles/types'

ProgressA11yOptions‚Äã

interface ProgressA11yOptions {
  max?: number
  min?: number
  now?: number
}

A11yProps‚Äã

interface A11yProps {
  'aria-valuemax': number
  'aria-valuemin': number
  'aria-valuenow': number
  role: 'progressbar'
}

ProgressProps‚Äã

interface ProgressProps {
  bar: {
    ...A11yProps
    className: string
    style: {
      width: number
    }
  }
  wrapper: { className: string }
}

JSProgressProps‚Äã

interface JSProgressProps {
  bar: {
    ...A11yProps
    cssProps: TemplateLiteralArray
    styles: Record<string, unknown>
  }
  wrapper: {
    cssProps: TemplateLiteralArray
    styles: Record<string, unknown>
  }
}
